General Use For Checkout And Return

1) Use the Blue "Select Action" button to choose either Checkout or Return

2) Enter The Serial Number or PCN (or both) into the respective fields. These numbers should be found on the device you are Checking Out/Returning

3) The table will filter results as characters are entered. When the last digit of the PCN/Serial is entered, there should only be one item displayed in the table

-Ensure the the PCN/Serial Number is entered EXACTLY as shown in the table

-If no entry is shown in the table, the item is not in the database and must be added by an administrator

4) Fill out the remaining fields of the form (Either the Checkout or Return item Form), and click the Checkout/Return Item Button

-Checkout Fields: Location the item is being taken to, Your Name, Today's Date

-Return Fields: Today's Date

5) A message will display above the table stating whether the operation Succeeded or Failed.

How to Add New Items To the Inventory List

1) Go to the Database Management Tab

2) Use the Blue "Select Action" button to choose "Create New Entry"

3) Enter all of the required information into the form and click "Insert"

-Check the status message (above the table) to see whether the action succeeded or failed

How to Update Information for an Existing Inventory Entry

1) Go to the Database Management Tab

2) Use the Blue "Select Action" button to choose "Update New Entry"

3) Enter the PCN/Serial Number (either one or both) for the item entry you want to update

-The table will filter results as you enter characters. After entering the last character, there should only be 1 item in the table

-Ensure that the PCN/Serial Number(s) are entered exactly as shown in the table

4) Enter the information you wish to change into the rest of the form.

-Any fields left blank will be left unchanged

-Click "Update" once you are done entering the new information

-Check the status message above the table to see if the update succeeded or failed.

How to Delete Entries From the Database

1) Go to the Database Management Tab

2) Use the Blue "Select Action" button to choose "Update New Entry"

3) Enter the PCN/Serial Number(Either one or both) for the item we want to delete

-Click Delete and check the status message to see if it succeeded or failed

How to Backup the Database

1) Go to the Database Management Tab

2) Use the Blue "Select Action" button to choose "Backup Inventory"

3) Enter a name for the backup file to be saved as

4) Click "Backup" and Save the file to your computer

How to Restore the Database

Note: This is probably a very rare task

Only use this if you have made a serious error in editing the database and want to restore it to a previous form


1) Go to the Database Management Tab

2) Use the Blue "Select Action" button to choose "Restore Inventory"

3) Click "Browse" and select the CSV file (obtained from the backup inventory section)

-Click Restore
